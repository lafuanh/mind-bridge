# Brain Map — 3-Brain Architecture

## The System

```
                    ┌─────────────────┐
                    │   Brain 01      │
                    │   Biological    │
                    │   (always on)   │
                    └────────┬────────┘
                             │ raw thought
                             ▼
                    ┌─────────────────┐
                    │   Brain 02      │
                    │   Notion /dump  │
                    │   (capture)     │
                    └────────┬────────┘
                             │
                      ┌──────▼──────┐
                      │   CLAUDE    │
                      │   bridge    │
                      │  3x daily   │
                      └──┬───────┬──┘
                         │       │
           ┌─────────────▼──┐  ┌─▼──────────────┐
           │  Notion         │  │  Obsidian       │
           │  what you DO    │  │  what you KNOW  │
           │                 │  │                 │
           │ Tasks/          │  │ Knowledge/      │
           │ Projects/       │  │ Research/       │
           │ Life/           │  │ English/        │
           │ Blog pipeline/  │  │ Life/journal/   │
           │ Master review   │  │ Output/         │
           └─────────────────┘  └─────────────────┘
                    │
          ┌─────────▼─────────┐
          │  Google Calendar  │
          │  what has a TIME  │
          └───────────────────┘
```

## Sync Windows

| Time (WIB) | Sync | Depth |
|-----------|------|-------|
| 05:00 | Morning brief | Short — goals + plan |
| 11:45 | Midday flush | Quick — clear chaos |
| 22:00 | Night digest | Full — classify + reflect |

- `/dump` — Brain 02 capture file
- `Master review` — live dashboard
- `Tasks/` — daily planner database
- `Projects/` — active + queued projects
