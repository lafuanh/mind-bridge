# Claude Commands — Your Life OS

These are quick commands you can say to Claude in any session.
Claude reads CLAUDE.md first, then executes the command.

## Sync commands
- `run morning brief` — simulate morning sync
- `run midday flush` — simulate midday sync
- `run night digest` — full night classify + route

## Review commands
- `show master page` — today's command center
- `language drill` — 5 vocab + 1 mistake + 1 rewrite
- `blog check` — review flagged blog entries
- `show graph` — describe current Obsidian connections

## Write commands
- `obsidian write [topic]` — new note in correct folder
- `write journal [date]` — daily journal from dump
- `write vocab [date]` — vocab note from dump

## Archive commands
- `archive notion` — guide quarantine of old Notion folders
- `archive obsidian` — move old notes to _archive/

## Build commands
- `show roadmap` — current project status
- `next step` — what to build next in your-os
