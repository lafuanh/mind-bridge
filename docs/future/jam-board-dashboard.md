# Future: Jam Board Dashboard

**Status:** Planned
**Priority:** Medium

## Concept

A visual dashboard that shows your Life OS state at a glance — like a jam board / kanban view of your current goals, active projects, today's schedule, and knowledge graph growth.

## Possible Features

- Today's goals and progress (from Daily Planner)
- Active project cards with last update timestamp
- Blog pipeline status (flagged → approved → published)
- Obsidian vault growth metrics (notes added this week, new backlinks)
- Habit streak tracker
- Weekly reflection summary
